Web Designer Nozomi Matsuo

東京都在住のWebデザイナー、まつおのぞみです。デザイナーとしての仕事の幅を広げる為、オンラインスクールでWeb制作について学んでいます。これからさらにWeb制作のスキルを身につけて、個人事業主向けにWebサイト運用のアドバイスもできるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript、PHP、WordPressも現在オンラインスクールにて勉強中です。

Paper design

一瞬で魅力を伝える名刺やフライヤー制作が得意です。シンプルなデザインからデコラティブなものまで、お客さま層に合わせてフルオーダーメイドで制作します。

Fashion

アパレル業界で働いていた経験から、ブランディングをベースにデザインをご提案します。必要な場合はファッションのご提案もします。
